Introduction


Objective

Upon completion of this lesson you will be able to identify the major components of the Audacity interface with 100 percent accuracy.

Purpose

This module prepares users for further instruction in the use of Audacity by familiarizing them with the interface layout, tools, and controls. Upon completion, users should be able to recognize and select the appropriate parts of the interface when asked.
